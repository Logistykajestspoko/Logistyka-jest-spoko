# Logistyka-jest-spoko
Logistyka jest spoko. Ale zawsze może być spokojniejsza. Zrób audyt w swojej firmie. Niech ktoś z zewnątrz wejdzie, posiedzi, popatrzy i zda Ci raport. Jak Twoja logistyka dokładnie działa i dlaczego może lepiej. Czy możesz robić to taniej. A może szybciej. A może dokładniej. Czy jesteś w XXI czy w IXX wieku w Twoich procesach. Powiem, bez ściemy.
